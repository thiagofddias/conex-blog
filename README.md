## Description

This is a project done in a course to build an API using GraphQL, NestJS and Prisma ORM. In this code, it is a complete CRUD API of a blog

## Project setup

```bash
npm install
```

## Compile and run the project

development
```bash
npm run start
```

watch mode
```
npm run start:dev
```

production mode
```
npm run start:prod
```

## Run tests

unit tests
```bash
npm run test
```

e2e tests
```
npm run test:e2e
```

test coverage
```
npm run test:cov
```
